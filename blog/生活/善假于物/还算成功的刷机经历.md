## 刷入twrp，类似win10重装的pe。
1. 解锁后进入fastboot模式刷入twrp。然后马上进行第二步。
2. 第一步成功后，立刻进入recovery模式。如果这时不小心重启，就会被miui自带的recovery覆盖。就要重复第一步。
3. 进入twrp后，选择wipe->formatdata->yes。
4. 使用adb工具传输刷机包到根目录下。
```
adb push axxx.zip /
```
5. 然后点击install，选择根目录的刷机包axxx.zip，开始安装。
6. 成功后重启即可。
