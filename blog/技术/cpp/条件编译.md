# 条件编译

编译器根据条件的真假决定是否编译相关代码

## 常见条件编译的方法

1. 根据宏是否定义
  
  ```c
  #ifdef <macro>
  ...
  #else
  ...
  #endif
  ```
  

举个例子

```c
  1 #include <stdio.h>
  2 #define _DEBUG_
  3 int main(int argc,const char* argv[])
  4 {
  5 #ifdef _DEBUG_
  6         printf("Hello World\n");
  7 #else
  8         printf("Welcome to my blog!\n");
  9 #endif
 10         return 0;
 11 };
 10 }
```

这里的`_DEBUG_`定义了，所以打印`Hello World`
