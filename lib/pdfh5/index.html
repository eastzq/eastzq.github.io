pdfh5.js - 移动端预览PDF插件，基于pdf.js和jQuery

本插件所需文件如下：

  1. cmaps文件夹-解决字体不全问题（下载压缩包解压） 下载大小：969kb
  2. pdf.js 下载大小：317kb
  3. pdf.worker.js 下载大小：749kb
  4. jquery-2.1.1.min.js 下载大小：94kb
  5. pdfh5.js 下载大小：22kb
  6. pdfh5.css 下载大小：9kb

pdfh5支持在线预览，修改地址栏file=后的地址就行，如果不填写，就在实例化的时候传入pdf地址：

http://www.gjtool.cn/pdfh5/pdf.html?file=//www.gjtool.cn/pdfh5/test.pdf

默认优先获取浏览器地址栏？file=后面的地址，如果地址栏没有，再拿配置项的pdfurl或者data来渲染pdf。pdfh5实例初始化：

    var pdfh5 = new Pdfh5('.pdfjs', {
        pdfurl: 'default.pdf'
    });
